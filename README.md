# Python Automation Framework (Testiranje Performansi)

Ovaj projekat služi za automatizovano testiranje logike i performansi Python aplikacija.

### Šta ovaj projekat pokriva?
- **Unit testovi**: Osnovna logika i parametri (Pytest)
- **Mocking**: Simulacija eksternih API poziva (`unittest.mock`)
- **Performance Monitoring**: Merenje RAM-a, CPU-a i vremena (`psutil`)
- **Reporting**: HTML izveštaji i SVG grafikoni performansi

### Kako pokrenuti?
1. Instalirajte zavisnosti: `pip install -r requirements.txt`
2. Pokrenite sve testove: `./run`
